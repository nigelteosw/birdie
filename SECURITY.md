# Security Policy

## Supported versions

Until Birdie publishes versioned releases, only the current `main` branch receives security fixes.

## Reporting a vulnerability

Do not open a public issue. Use [GitHub's private vulnerability reporting flow](https://github.com/nigelteosw/birdie/security/advisories/new) for this repository and include the impact, reproduction steps, affected configuration, and any suggested mitigation.

Please avoid accessing or modifying data that does not belong to you while investigating a potential issue. Maintainers will acknowledge the report through the private advisory and coordinate disclosure there.
